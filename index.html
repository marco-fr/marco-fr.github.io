Chess Engine

C++

My engine is about 1500 rated. I connected it to lichess and it can beat Stockfish level 5. The ideas for the techniques I used pretty much all came from the chess programming wiki. Right now, I have implemented minimax with alpha-beta pruning, bitboards, magic bitboards, transposition table, null-moves and quiescence.

6502 Emulator

C++

The 6502 is a processor from the 1970s. I implemented all 12 addressing modes, 6 registers, and 151 opcodes. There is not really a use for writing an emulator like I did by itself because a CPU can't do much alone.
